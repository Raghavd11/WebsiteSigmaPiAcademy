DATA SCIENCE

Semester IV - Detailed Syllabus

Unit Topics
Unit 4: Decision Tree Learning and Unsupervised Learning
4.1: Decision Trees
• Concept of entropy for measuring purity/impurity of training examples
• Information gain as a measure of attribute effectiveness (basics + equation)
• Inducing decision trees using ID3 algorithm, illustrative example
• Overfitting and reduced error pruning
• Discretizing continuous attributes using information gain (binary split only)
• Difference between supervised and unsupervised learning

4.2: Unsupervised Learning and Clustering
• What is unsupervised learning?
• Supervised vs Unsupervised learning
• What is clustering?
• Why clustering is an unsupervised technique
• Real-world applications of clustering
• Clustering vs Classification
• K-means clustering algorithm with simple use cases
Unit 5: Data Visualization Technique
• Need for data visualization
• Visualization for small number of attributes: Stem and Leaf Plots, 1D/2D Histogram, Box Plots, Pie Chart, Scatter Plots
• Visualizing spatio-temporal data: Contour plots, Surface plots
• Visualizing higher-dimensional data: Data matrix plots
• Heatmap visualization
• Introduction to visualization platforms: Tableau, Google Chart
Unit 6: Artificial Neural Network
• Biological motivation for ANN
• Simple neuron model: McCulloch & Pitts (1943)
• Activation functions: Threshold, Sigmoid
• Perceptron as linear classifier, perceptron training rule
• AND/OR using threshold perceptron, linear separator equation
• Delta rule for unthresholded perceptron
• Hidden layers and XOR example
• Need for non-linearity
• Network structures: Feedforward and Recurrent Networks (basic only)
• Training feedforward neural networks using Backpropagation (concepts only)
• Generalization, overfitting, stopping criteria
• Using validation data to avoid overfitting
• ANN architecture for handwritten digit recognition (input/output representation + block diagram)
• Need for automatic feature learning
• Conventional FFNN vs CNN
• Convolution layer role, example of 2D convolution
• Function of pooling layer
• CNN block diagram for handwritten digit recognition
Unit 7: Case Studies in Data Science
• Weather forecasting using statistical and ML tools (based on covered algorithms)
• Sentiment analysis using machine learning (based on covered algorithms)
• Simple collaborative filtering-based recommendation system
