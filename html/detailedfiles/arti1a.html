Semester I - Detailed Syllabus

Unit Topics
Unit 1: Computer Fundamental
1.1:
•History of Computers, Basic Computer Hardware, Input and Output Devices, Basic Computer Architecture, Memory and CPU, Networking Overview (LAN, MAN, WAN, Internet, WiFi, etc.), Types of Computers (Workstation, Desktop, Smartphone, Embedded Systems, etc.), Overview of Software (System Software and Application Software with Example Names), Definition and Functions of Operating Systems (Examples: Windows, Linux, Android, etc.).
1.2:
•Bit, Byte and Word, Number System (Base, Binary, Decimal, Octal, Hexadecimal), Conversion Between Number Systems, Boolean Logic and Gates, Boolean Operators (OR, AND, NOT), ASCII Code, Concept of Algorithm and Flowchart.
1.3:
•Basics of Computer Programming (High-Level Language, Assembly Language, Machine Language — Definitions and Block Diagrams), Overview of Compiler and Interpreter (Definitions, Examples of Compiled Languages like C, C++ and Interpreted Languages like Python), Procedural vs Object-Oriented Programming (Key Features, Basic Differences, and Examples: BASIC, FORTRAN, C, C++, Java, Python).
Unit 2: Introduction to Python Programming
2.1:
•Basics of Python Programming (Hello World Example, Writing and Running Code), Print Statement, Concept of Class and Object, Data Types (Integer, Float, String), Variables, Operators (Assignment, Logical, Arithmetic), Input from Console, Conditional Statements (If-Else, Nested If-Else), Collections (List, Tuple, Set, Dictionary), Loops (For, While, Nested Loops), Iterator, String Operations (Compare, Concatenation, Substring), Functions and Recursion.
2.2:
•Overview of Linear and Non-Linear Data Structures (Definitions, Schematic Views, Differences), Arrays (1D, 2D, Relation to Matrix), Array Operations (Access by Index, Insert, Delete, Search), Stack (LIFO Concept, Push, Pop, Peek, Size), Queue (FIFO Concept, Enqueue, Dequeue, Peek, Size), Python List Methods for Stack/Queue Operations, Overview of NumPy and Basic Array Operations (arrange(), shape(), ndim(), dtype(), etc.), Binary Tree (Definition and Schematic View Only).
2.3:
•Linear Search and Binary Search Algorithms, Bubble Sort Algorithm.
Unit 3: Introduction to Linear Algebra
•Basic Matrix Operations: Addition, Subtraction, Multiplication, Transpose, Identity Matrix.
•Introduction to Vectors: Unit Vector, Normal Vector, Euclidean Space.
•Probability Concepts: Distribution, Frequency, Mean, Median, Mode, Variance, Standard Deviation, Gaussian Distribution.
•Distance Functions: Euclidean Norm, Distance Between Two Points in 2D and 3D, and Generalization to n-Dimensions.
