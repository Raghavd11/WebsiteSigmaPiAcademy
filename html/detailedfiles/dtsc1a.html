DATA SCIENCE

Semester I - Detailed Syllabus

Unit Topics
Unit 1: Computer Fundamentals
1.1: Basic Concepts
• History of computer
• Basic Computer hardware
• Input and output devices
• Basic computer architecture: memory and CPU
• Networking overview: LAN, MAN, WAN, Internet, Wi-Fi
• Types of computers: workstation, desktop, smartphone, embedded system
• Overview of software: system vs. application (mention names only)
• Definition and functions of Operating System (Windows, Linux, Android, etc.)

1.2: Number Systems & Logic
• Bit, Byte and Word
• Number Systems: Base, Binary, Decimal, Octal, Hexadecimal
• Number system conversions
• Boolean logic and gates
• Boolean operators: OR, AND, NOT
• ASCII code
• Concept of Algorithm and Flowchart

1.3: Computer Programming Basics
• Programming levels: High-level, Assembly, Machine language (with block diagrams)
• Compiler vs Interpreter (mention languages like C, C++, Python)
• Procedural vs Object-Oriented Programming (BASIC, FORTRAN, C vs C++, Java, Python)
Unit 2: Introduction to Python Programming
2.1: Python Basics
• Hello World program, writing and running code
• Print statement
• Class and object concepts
• Data-types: integer, float, string
• Variables and operators (assignment, logical, arithmetic)
• Console input
• Conditional statements: if-else, nested if-else
• Collections: List, Tuple, Set, Dictionary
• Loops: for, while, nested loops
• Iterators
• String operations: compare, concatenate, substring
• Functions and recursion

2.2: Data Structures and NumPy
• Overview of linear vs non-linear data structures
• Arrays (1D, 2D) and matrix relation
• Basic operations: access, insert, delete, search
• Stack (LIFO): push, pop, peek, size
• Queue (FIFO): enqueue, dequeue, peek, size
• Using Python List methods for stack/queue/array
• NumPy basics: arrange(), shape(), ndim(), dtype()
• Binary Tree: definition and schematic overview

2.3: Searching and Sorting
• Linear Search
• Binary Search
• Bubble Sort
Unit 3: History of AI & Linear Algebra
3.1: History of AI
• Alan Turing and Enigma
• Mark I machines
• 1956 – Birth of the term AI
• AI Winter of the 1970s
• Expert systems of the 1980s
• Journey to present-day AI
• Distinctions between AI, Pattern Recognition, and Machine Learning
Note: This section should be taught as a narrative involving WWII, Enigma, and modern computers

3.2: Linear Algebra Fundamentals
• Matrix operations: addition, subtraction, multiplication, transpose, identity matrix
• Introduction to vectors: unit vector, normal vector
• Euclidean space
