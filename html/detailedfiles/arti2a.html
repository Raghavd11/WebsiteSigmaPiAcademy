ARTIFICIAL INTELLIGENCE

Semester III - Detailed Syllabus

Unit Topics
Unit 1: Foundation of Statistics for Machine Learning
Distance between Distributions – Euclidean Norm, Pearson Correlation Coefficient, Basic Concepts of Chi-Square Distance, Bayes' Theorem, and Bayesian Probability.
• Real n-dimensional space (ℝn) and Vector Algebra, Dot Product, and Vector Projections.
• Product Moment Correlation Coefficient (Pearson’s Coefficient) and its Use in Measuring Relationship Between Data Sets.
• Chi-Square and Its Use in Comparing Distributions.
• Conditional Probability and Bayes’ Theorem, Concept of Conditional Independence.
Unit 2: Introduction to Machine Learning
2.1 :
• What is Machine Learning?
• Difference Between Traditional Programming and Machine Learning
• Machine Learning and Its Relation to AI
• Applications of Machine Learning
• Why Machines Need to Learn
• Types of Machine Learning: Supervised, Unsupervised, Semi-Supervised, Reinforcement Learning
• Linear Regression with One Variable
• Hypothesis Representation and Hypothesis Space
• Learning Requires Bias
• Concept of Training Examples
• Loss Function and the Idea of Training as Iterative Trial-and-Error
• Disadvantages of Iterative Training
• Mean Squared Error (MSE), Gradient Descent Algorithm
• Effect of Learning Rate
• Importance of Feature Scaling (Min-Max Normalization)

2.2 :
• What is a Feature or Attribute? Examples and Types (Continuous, Categorical)
• Representation of Training Examples with Multiple Features
• Linear Regression with Multiple Attributes (Formula Only)
• Feature Cross and Polynomial Regression
Unit 3: Supervised Learning
3.1 :
• Difference Between Regression and Classification
• Real-World Classification Problems
• Linear Classification and Threshold Classifier
• Misclassification Error, Accuracy
• Input Space and Linear Separators
• Limitations of Threshold Classifier
• Logistic Regression Model (Without Derivation)
• Use of Logistic Function in Hypothesis Definition
• Probabilistic Interpretation of Logistic Regression Output
• Binary Classification Using Logistic Regression
• Multiclass Classification Using One-vs-All Strategy
• Instance-Based Learning and K-Nearest Neighbor (KNN) Classifier
• Curse of Dimensionality

3.2 :
• Measuring Classifier Performance: Confusion Matrix
• True Positive, True Negative, False Positive, False Negative
• Error, Accuracy, Precision, Recall, F-Measure
• Sensitivity and Specificity
• K-Fold Cross Validation
