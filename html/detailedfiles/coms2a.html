COMPUTER SCIENCE

Semester III - Detailed Syllabus

Unit Topics
Unit 1: Python Programming
1.1: Introduction to Python
• Features of Python
• Executing a “Hello World” program
• Interactive mode vs Script mode
• Python character set, tokens (keyword, identifier, literal, operator, punctuator)
• Variables, l-value and r-value, comments

1.2: Data Types
• Number (int, float, complex), Boolean, Sequence (string, list, tuple), None, Dictionary
• Mutable vs Immutable types

1.3: Operators
• Arithmetic, Relational, Logical, Assignment, Augmented Assignment
• Identity (is, is not), Membership (in, not in)

1.4: Expressions & I/O
• Operator precedence, expression evaluation
• Type conversion: implicit and explicit
• Input/output functions

1.5: Errors
• Syntax errors, logical errors, run-time errors

1.6: Flow of Control
• Indentation, sequential, conditional, and iterative flow

1.7: Conditional Statements
• if, if-else, if-elif-else
• Flowcharts and examples (absolute value, sort numbers, divisibility)

1.8: Iteration
• for loop, while loop, range(), break, continue
• Nested loops and pattern generation
• Examples: series summation, factorial

1.9: Strings
• Operations: concatenation, repetition, membership, slicing
• Loop traversal
• Methods: len(), capitalize(), lower(), upper(), count(), find(), index(), startswith(), endswith(), isalnum(), isalpha(), isdigit(), islower(), isupper(), isspace(), strip(), replace(), join(), partition(), split()

1.10: Lists
• Indexing, slicing, operations
• Traversal with loops
• Methods: append(), extend(), insert(), count(), index(), remove(), pop(), reverse(), sort(), min(), max(), sum()
• Nested lists, frequency count, linear search

1.11: Tuples
• Indexing, operations, assignment
• Methods: len(), tuple(), count(), index(), min(), max(), sum()
• Nested tuples

1.12: Dictionaries
• Access using keys, mutability
• Traversal
• Methods: keys(), values(), items(), get(), update(), del, clear(), copy(), pop(), popitem(), setdefault(), max(), min(), sorted()

1.13: Python Modules
• Importing modules using import and from
• math (pi, e, sqrt(), ceil(), floor(), pow(), fabs(), sin(), cos(), tan())
• random (random(), randint(), randrange())
• statistics (mean(), median(), mode())

1.14: Functions
• Built-in, module-defined, user-defined
• Parameters: positional, default
• Return values, scope (local/global)

1.15: Exception Handling
• try, except, finally blocks
Unit 2: E-Commerce
2.1: Introduction to E-Commerce
• Definition, main activities, goals
• Technical components, functions, advantages/disadvantages
• Scope and applications
• Comparison with E-Business
• Types: C2C, C2G, G2G, B2G, B2P, B2A, P2P, C2A, B2B, B2C
• Internet, Intranet, Extranet
• Role of Internet in B2B
• Web Promotion: banners, exchanges, shopping bots

2.2: Electronic Data Interchange (EDI)
• Concepts, limitations, applications, disadvantages, EDI model

2.3: Electronic Payment Systems
• Introduction, types, payment types
• Value exchange system
• Credit card system, Electronic Fund Transfer
• Paperless billing, modern payments, e-cash

2.4: Internet Marketing
• Pros and cons of online shopping
• Justification of internet businesses
• Marketing techniques, E-cycle of marketing
• Personalization in e-commerce
