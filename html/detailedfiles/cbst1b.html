CYBER SECURITY

Semester II - Detailed Syllabus

Unit Topics
Unit 4: Network Security
4.1: Overview and Importance
• Introduction to Network Security and its Importance

4.2: Network Access Control
• Authentication Mechanisms: Passwords, Biometrics, Hardware Tokens
• Authorization and Access Control Lists (ACLs)

4.3: Firewalls
• Role of Firewalls in Network Security
• Types: Packet-Filtering, Stateful Inspection
• Architectures: Host-Based, Network-Based Firewalls
• Configuration and Management (Linux IPTables)
• Network Address Translation (NAT)

4.4: Intrusion Detection Systems (IDS)
• Overview and Importance
• Types: Host-Based IDS, Network-Based IDS
• IDS Architectures: Centralized, Distributed
• Detection Techniques: Signature-Based, Statistical Anomaly Detection
• IDS Management: Sensor Configuration and Snort Rules

4.5: Wireless Network Security
• Vulnerabilities in Wireless Networks
• WPA2 and WPA3 Security Standards

4.6: IoT Security
• Security Threats in IoT Devices
• Attack Vectors: Device Compromise, Data Interception, DoS
• Case Studies of IoT Security Breaches
• Privacy Issues in IoT Deployments

4.7: Cloud Security
• Security Threats in Cloud Environments
• Security Measures for Cloud Storage
Unit 5: Cryptography – Part I
5.1: Introduction
• Encryption & Decryption
• Plain Text, Cipher Text
• Symmetric Cipher Models: Substitution, Transposition Ciphers
• Steganography

5.2: Secret Key Cryptography
• Symmetric Key Encryption
• Block Cipher Structures
• DES: Example and Strength
• AES: Example and Strength
• Block Cipher Modes of Operation
• Stream Ciphers: Synchronous, Asynchronous
• Autokey and RC4 Stream Cipher

5.3: Public Key Cryptography
• Principles of Public Key Cryptography
• RSA Algorithm with Examples
Unit 6: Internet Security
6.1: Social Engineering
• Overview and Importance
• Common Techniques: Phishing, Pretexting, Baiting, Vishing, Smishing
• Impersonation
• Case Studies: Banking Frauds, Social Media Fraud, Fake Profiles & Videos
• Best Practices for Prevention

6.2: Email Security
• Email Threats: Spam, Spoofing, Phishing, Spear Phishing, Malware, Credential Theft
• Email Security Best Practices
