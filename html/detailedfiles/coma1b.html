COMPUTER APPLICATION

Semester II - Detailed Syllabus

Unit Topics
Unit 4: Data Structure
Definition, types of data structure-linear and non-linear
Arrays:
•1D, 2D and their applications.
Linked List:
•Basic concepts of Single, circular and double link list.
Stack:
•Stack operations (push and pop), applications of Stack.
Queue:
•Queue operations, applications of queue, basic concepts of circular queue and priority queue.
Recursion:
•Definition, advantages and limitations of recursion.
Searching & Sorting:
•Linear Search, Binary Search, and their comparison.
•Bubble Sort and its Implementation.
Unit 5: Computer Networks
Introduction to Networking:
•Analogue and digital Communication
•Mode of Communication- Simplex, half duplex and full duplex
•Network Architecture- Client server, Peer to Peer
•Serial and Parallel Communication
•Measuring Capacity of Communication Media (bandwidth, channel capacity, baud)
•Synchronous and asynchronous Transmission Mode
•Baseband and Broadband network.
Transmission Media:
•Wired Communication Media (Twisted Pair, Co-axial cable, Fiber Optic).
Communication Media (Radio wave, Microwave, Infrared, Satellite).
Network Connecting Devices:
•Modem, Ethernet Card, RJ45, Repeater, Hub, Switch, Router, Gateway, Wifi card.
Network Type and Topologies:
•Types of Network-LAN, MAN, WAN.
Network Topologies- Bus, Star, Ring, Tree.
Network Protocols:
•HTTP, FTP, PPP, SMTP, TCP/IP, POP3, TELNET, HTTPS, VolP.
Referential Model:
•OSI Model (Basic Concept, use of devices and protocols at different layers).
Introduction to Web Services:
•WWW, HTML, XML, IP Addresses, Domain names, URL, ISP, Website, Web browser, Web Server, Web Hosting.
HTML
Unit 6: Ethics
Digital Footprints
Data Protection:
•Definition, importance of data protection, data privacy, data security.
Cyber Crime:
•Definition, hacking, eavesdropping, phishing and fraud emails, ransomware, cyber trolls, cyber bullying.
Cyber safety:
•Safely browsing the web, identity protection, confidentiality.
Malware:
•Viruses, trojans, adware.
E-waste management:
•Proper disposal of used electronic gadgets.
