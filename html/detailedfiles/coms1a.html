COMPUTER SCIENCE

Semester I - Detailed Syllabus

Unit Topics
Unit 1: Computer System and Organisation
1.1: Basic Computer Organisation
• CPU, Primary Memory (RAM, ROM, Cache)
• Secondary Storage Devices
• Input/Output Devices
• Units of Memory (bit, byte, KB, MB, GB, TB, PB)

1.2: Classification of Computers
• Supercomputer
• Mainframe
• Mini Computer
• Personal Computer (PC)

1.3: Concepts of Software
• Definition of Software
• Types of Software:
   ○ System Software: Translators (Assembler, Interpreter, Compiler), Loader, Linker
   ○ Operating System: Definition, Functions, Types (Single User, Multiuser, Multiprogramming, Multiprocessing, Time Sharing)
   ○ Application Software (Definition, Examples)
   ○ Utility Software
• GUI vs. CUI (Examples using LINUX - Basic Commands)

1.4: Number System
• Binary, Octal, Decimal, Hexadecimal
• Conversion Between Number Systems
• Weighted Code: BCD, 8421
• Non-weighted Code: GREY, Excess-3
• Encoding Schemes: ASCII, ISCII, Unicode
• 1’s and 2’s Complement

1.5: Boolean Algebra
• Boolean Postulates
• Logic Gates: NOT, AND, OR, NAND, XOR, XNOR
• Truth Tables
• De Morgan’s Theorems
• SOP, POS Forms
• Simplification using K-Map and Boolean Algebra
• Logic Circuit Design
Unit 2: Programming Fundamentals
2.1: Concept of Programming
• Instruction and Program (Definition and Examples)
• Programming Languages: High-level, Low-level, Assembly
• Procedural vs. Non-Procedural Programming
• Structured Programming
• Object-Oriented Programming (OOP)

2.2: Algorithm Fundamentals
• Definition and Characteristics
• Recursive and Non-Recursive Algorithms
• Representation: Flowchart, Pseudocode
• Efficiency: Space and Time Complexity
• Asymptotic Notations: Big O, Big Ω, Big Θ

2.3: Introduction to Problem Solving
• Steps: Problem Analysis, Algorithm Design, Coding, Testing, Debugging
Unit 3: Introduction to C
3.1: Basic Structure of C
• Character Set
• Keywords, Identifiers
• Constants, Variables, Type Declaration
• Sample Programs
• Pre-processor

3.2: Operators
• Arithmetic, Relational, Logical, Assignment
• Increment/Decrement, Conditional, Comma
• Operator Precedence and Associativity
• Expression Evaluation, Type Conversion
• Character I/O, Escape Sequences, Formatted I/O

3.3: Branching and Looping
• if, if-else
• while, do-while, for

3.4: Arrays and Structures
• One-Dimensional and Two-Dimensional Arrays
• String Handling: Reading, Writing, Concatenation, Comparison, String Functions
• Structures: Initialization, Arrays of Structures, Arrays within Structures, Nested Structures

3.5: User Defined Functions
• Need for Functions
• Call by Reference vs. Call by Value
• Return Types, Nesting, Recursion

3.6: Pointers
• Declaration and Initialization
• Pointer Operators and Arithmetic
• Accessing Variables
• Pointers with Arrays, Strings, and Functions
