COMPUTER SCIENCE

Semester II - Detailed Syllabus

Unit Topics
Unit 1: Data Structure
1.1: Introduction
• Definition and types of Data Structure: Linear and Non-Linear
• Abstract Data Types (ADT)

1.2: Arrays
• One-Dimensional and Two-Dimensional Arrays
• Applications of Arrays

1.3: Linked Lists
• Single Linked List
• Circular Linked List
• Doubly Linked List

1.4: Stack
• Stack Operations: Push and Pop
• Implementation using Array and List
• Applications of Stack

1.5: Queue
• Queue Operations
• Implementation using Array and List
• Circular Queue
• Dequeue
• Priority Queue

1.6: Recursion
• Definition
• Advantages and Limitations
• Internal Stack Mechanism
• Tail Recursion

1.7: Searching and Sorting
• Linear Search vs. Binary Search
• Bubble Sort and its Implementation
Unit 2: Computer Networks
2.1: Introduction to Networking
• Analog and Digital Communication
• Modes: Simplex, Half Duplex, Full Duplex
• Network Architecture: Client-Server, Peer-to-Peer
• Serial and Parallel Communication
• Bandwidth, Channel Capacity, Baud
• Synchronous vs. Asynchronous Transmission
• Baseband and Broadband Networks

2.2: Transmission Media
• Wired: Twisted Pair, Co-axial Cable, Fiber Optic
• Wireless: Radio Wave, Microwave, Infrared, Satellite

2.3: Network Connecting Devices
• Modem, Ethernet Card, RJ45
• Repeater, Hub, Switch, Router
• Gateway, WiFi Card

2.4: Network Types and Topologies
• Network Types: LAN, MAN, WAN
• Topologies: Bus, Star, Ring, Tree

2.5: Network Protocols
• HTTP, FTP, PPP, SMTP, TCP/IP, POP3, TELNET, HTTPS, VoIP

2.6: Referential Models
• OSI Model: Basic Concepts
• Use of Devices and Protocols at Different Layers

2.7: Web Services
• WWW, HTML, XML
• IP Addresses, Domain Names, URL
• ISP, Website, Web Browser, Web Server, Web Hosting
Unit 3: Ethics
3.1: Digital Footprints

3.2: Data Protection
• Intellectual Property Rights (IPR): Copyright, Patent, Trademark
• Violations: Plagiarism, Copyright Infringement, Trademark Infringement
• Open-Source Software and Licensing: Creative Commons, GPL, Apache

3.3: Cyber Crime
• Definition
• Hacking, Eavesdropping, Phishing, Fraud Emails
• Ransomware, Cyber Trolling, Cyber Bullying

3.4: Cyber Safety
• Safe Web Browsing
• Identity Protection
• Confidentiality

3.5: Malware
• Viruses, Trojans, Adware

3.6: E-Waste Management
• Proper Disposal of Electronic Gadgets

3.7: Information Technology Act (IT Act)
